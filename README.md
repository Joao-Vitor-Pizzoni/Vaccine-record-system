# Vaccine Record System

This C project manages vaccination records using stacks and queues, both static and dynamic.

The system allows users to insert, remove, and view vaccination records based on LIFO (stack) and FIFO (queue) data structures.

It can be used in hospitals or clinics to efficiently organize vaccination records and ensure proper data management.

## Features

* Insert, remove, and view vaccination records
* Stack (LIFO) and queue (FIFO) implementation
* Supports static and dynamic memory allocation
* Simple console interface

## Technologies

* C Programming Language
* Data Structures: Stack, Queue

## Use Cases

* Hospital or clinic record management
* Educational project for practicing data struc
